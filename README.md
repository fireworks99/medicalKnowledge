# 扫描二维码查看网页



## 缘由

一个学医的朋友有这样的需求：科室宣教（告知患者如何挂号、如何办理住院、出院等手续），想将内容制作成网页，然后生成二维码，挂在科室。患者扫描二维码，即可查看响应的网页。



## 步骤

1. 将内容制作成网页（HTML、CSS、JS）
2. 将上述这些静态资源托管到 [Cloudflare Pages](https://pages.cloudflare.com/)：登录，找到`Workers & Pages` => `Create application` => `Looking to deploy Pages? Get started` => `Drag and drop your files` => 填写名称，上传静态资源文件。部署后会返回一个网址，例如 http://example.com/
3. 将网址写入二维码。这里有很多工具，我用的qrcode库，全局安装后，终端执行 `qrcode -o 1.png http://example.com/` 即可将网址写入二维码。



> 补充说明：用户使用设备扫描二维码后，设备开始解析。
>
> 如果解析出来的文本刚好是以 http / https 开头的字符串，那么设备自动唤起浏览器。
>
> 如果解析出来的文本只是普通文本，则提供其他选项（例如复制、添加到联系人、填做密码用于连接WIFI等）



## 技术

HTML里的图片采用 [Viewer.js](https://fengyuanchen.github.io/viewerjs/) 用于预览。静态资源托管采用 **Cloudflare Pages**，国内访问速度还可以，之前一直用的 **Github Pages** 或者 **vercel**，但他俩国内访问不了，或者能访问但很慢，另外国内的 Gitee Pages 暂停提供服务了，不然用它可能更好一些。
